# Housing Market Price Analysis

## Project Overview
This project analyzes housing market data to identify the factors that influence house prices.

## Dataset
The dataset was obtained from Kaggle and contains information such as area, number of bedrooms, bathrooms, parking, furnishing status, and other housing features.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Methodology
1. Loaded and explored the dataset
2. Cleaned categorical values
3. Checked missing values and duplicates
4. Performed descriptive statistics
5. Conducted univariate, bivariate, and multivariate analysis
6. Insights and recommendations

## Key Findings
- Area has a strong relationship with price
- Furnished houses tend to cost more
- More bedrooms and bathrooms often increase house value
- Some housing features contribute significantly to higher prices

## Conclusion
The analysis helps buyers and investors understand the factors that drive property prices and make better decisions.
